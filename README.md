# Expense Tracker

A full-stack expense tracking application built with Django, React, and PostgreSQL. This project was built as an introductory exercise to get hands-on experience with these technologies.

## Features

- **Add, Edit, and Delete Expenses** — Full CRUD support for managing your expenses
- **User Authentication** — Secure login and registration so each user manages their own data
- **Expense Categories** — Categorize expenses to keep things organized and easy to sort
- **Time-Based Filters** — Filter expenses by duration (e.g., Last 3 Months) to quickly review spending over a period

## Tech Stack

| Layer    | Technology          |
|----------|---------------------|
| Frontend | React               |
| Backend  | Django + Django REST Framework |
| Database | PostgreSQL          |

## What I Learned

- Setting up Django REST Framework with JWT authentication
- Connecting a React frontend to a Django backend via REST APIs
- Managing relational data with PostgreSQL
- Filtering querysets by date ranges in Django
